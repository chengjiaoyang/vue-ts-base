# 测试的一个项目的文件
##目录列表
* src ->
    * style -> less / scss 样式存放位置 
    * assets 静态资源存放位置
    * utils 工具方法(axios封装，全局方法等)
    * views 全部页面入口
        * page.vue vue 文件
        * page.less 当前组件的样式文件 
        * page.ts 当前组件 ts 文件
    * main.ts -> 入口文件    
    * router.ts -> 路由文件    
    * shims-tsx.d.ts -> 相关 tsx 模块注入    
    * shims-global.d.ts -> 相关 全局或者插件 模块注入    
    * App.vue -> 页面主入口
